阿里内推测试题目(关于生兔子问题)

发表于 2017-08-08 | 阅读次数

题目

1. 猎人把一对兔子婴儿(一公一母称为一对)放到一个荒岛上，两年之后，它们生下一对小兔，之后开始每年都会生下一对小兔。生下的小兔又会以同样的方式继续繁殖。
2. 兔子的寿命都是x(x>=3)年，并且生命的最后一年不繁殖。
3. 如果岛上的兔子多于10对，那么猎人会每年在兔子们完成繁殖或者仙逝之后，从岛上带走两对最老的兔子。请问y年(y>=3)后荒岛上所有的兔子加起来多少岁?(注意, 在条件3执行完之后)输入: 从命令行输入两行整数，第一行是x，第二行是y输出: y年后荒岛上所有的兔子岁数的总和

分析

由于是阿里内推时的笔试题目，没有给例子。这里我讲一下我的思路：
分别设f(x), p(x), t(x)表示x年之后，总的兔子数量，新出生的兔子数量，死去的兔子数量，这里数量都是多少对。
我简单推一下，假设兔子的寿命是3年，则如下：

时间(x年之后) f(x)(总数） p(x)(新出生数量） t(x)(死亡数量)
0 1 1 0
1 1 0 0
2 2 1 0
3 1 0 1
4 2 1 0
5 1 0 1
6 2 1 0
7 1 0 1
8 2 1 0
9 1 0 1
10 2 1 0

这里，0～10表示几年之后。题目说的也是几年之后（这里理解不知道对不对）。比如一开始，总的兔子数量是1对，也恰好就是新出生的一对。这里，我们可以简单得到几个递推式子：
$$ f(i) = f(i - 1) + p(i) - t(i) $$
$$ p(i) = f(i - 1) - p(i - 1) - t[i] $$
这里，$p(i)$表示新出生的，如果计算新出生的，这个很关键。因为，可以发现，先不考虑条件三，$t(x)$就是$p(x)$的平移，因为新生的兔子life年之后，寿命到了一定会挂掉。这里，根据题意，下一年挂掉的兔子一定是不能生小兔子，另外，上一年刚出生的兔子也是不能生兔子的(两年之后才能生小兔子，后面才一年生一次)，那么，只要把这两部分剪掉就行了。所以，$t[i]$表示$i$年之后，死去的兔子数量，$p[i-1]$表示上一年新生的兔子数量。$f(i - 1)$表上一年总的兔子数量，里面包含了上一年新出生的，和这年需要死去的，这两类兔子都不能生育。
$$ t[i] = p[i-life] $$
这里，死亡的兔子数量永远是新出生兔子数量序列的前几个, 也就是通过减去life去寻找。
其实，就是类似dp的一种算法去解决这个问题。
整个算法的空间复杂度最优是O(life)，代码中是O(n)，这一部分还可以优化。
