大疆2018校招软件岗位笔试题目(求有效工作时间)

发表于 2017-08-27 | 分类于 笔试 | | 阅读次数

题目

在DJI的工时打卡计算系统中，一个工作日的定义是由工作当日当天的凌晨03:00:00开始，到隔天凌晨03:00:00（不包含）为止。
这其中，工作日当天的中午12:30:00到下午14:00:00（不包含）为午休时间，不计入工作时间。
Tom由于工作繁忙，担心自己有时候会忘记打卡，因此他每次经过打卡机到时候都会不由自主地打一次卡。为了更好的了解自己的工作情况，Tom找到了自己整个八月份的乱序打卡记录，但是这些数据不小心弄乱了，所以Tom想请你帮个忙。从给出的N条记录中，找出所有有效的打卡工作日（即在该工作日中打开次数>=2并且有效工作时间>0)，并计算该工作日中的有效工作时间（按秒计算，除去打开时间内出现的午休时间）。其中工作日定义为当天3:00:00到次日2:59:59为当天工作日，即当天 2:59:59(包括）以前的打开记录当作前一天打开记录和有效工作时间，不计做当天的打开记录和有效工作时间。

输入

12
08.10 01:00:00
08.09 23:59:59
08.10 17:00:00
08.10 18:00:00
08.01 01:00:00
08.01 04:00:00
08.03 10:00:00
08.03 12:29:59
08.04 10:00:00
08.04 12:30:00
08.07 01:00:00
08.07 02:00:00

输出

08.03 8999
08.04 9000
08.06 3600
08.09 3601
08.10 3600

分析

这是大疆软件工程的笔试第二题目。整个题目没有啥难度，主要是要考虑很多情况。恰好博主没啥精神，昨天连续三场笔试，所以也就没有用力去求解，写了一部分就没兴趣再写了。想来，还是需要花费不少时间的。后来考试结束后又继续写，给大家做个参考。
算法:

  1. 对所有记录按照key和value进行对应。其中，key是”08.10”，value则是具体时间,单位为秒。
  2. 对每个key下的时间点进行排序，从大到小，然后大的时间减去小的时间，其中休息时间需要另外考虑。

问题:
这里面有一个问题比较复杂，就是如何输入数据。这里字符串中有空格，所以，不能采用平常cin输入，采用getline输入。但是，因为前面输入n后有一个回车，而getline函数是根据”\n”来判断是否读完一行的。我们这里采用先getchar()函数读取”\n”，然后不进行任何操作的方法来把这个问题规避掉。具体代码如下(代码还没有完全测试过所有case，哪位好心人可以考试碰到了测一下，作个反馈）。
